# Architecture Exports

This folder contains reusable source files and rendered assets for the MindPulse architecture diagrams.

## Contents

1. Mermaid source: `./mermaid/`
2. PlantUML source: `./plantuml/`
3. SVG exports: `./svg/`
4. PNG exports: `./png/`

## Regenerate

Run:

```bash
npm run export:architecture
```
